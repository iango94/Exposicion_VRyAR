# Exposicion\_VRyAR
- El Host (Moderador): Abre con el gancho, presenta el tema y hace las transiciones entre compañeros. Controla el tiempo con señas.

- Especialista AR (Realidad Aumentada): Explica el Uso Práctico 1.

- Especialista VR (Realidad Virtual): Explica el Uso Práctico 2.

- Especialista Mixto/Futuro: Explica el Uso Práctico 3 (el más disruptivo).

- El Visual/Demo: Encargado de que los videos o diapositivas fluyan perfecto. Si tienen un casco o un celular con una App de AR, él es quien - hace la demostración en vivo mientras los otros hablan.

- El "Cerrador": Responde la pregunta rápida del público y da la conclusión emocional.
